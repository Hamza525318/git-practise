git init -> Initialise an empty repository
git add file_name -> Add file to staging area
git add . -> Add all modified , untracked files to staging area
git commit -m "message" -> Commit the changes to the repository
git status -> Check the status of the repository
git log -> Check the commit history
git diff -> Check the difference between the working directory and the staging area
git diff --staged -> Check the difference between the staging area and the repository
git diff HEAD -> Check the difference between the working directory and the repository
git restore file_name -> Discard the changes in the working directory
git restore --staged file_name -> Unstage the file
git branch -> Check the current branch
git branch branch_name -> Create a new branch
git checkout branch_name -> Switch to a branch
git merge branch_name -> Merge a branch into the current branch
git remote add origin url -> Add a remote repository
git push origin branch_name -> Push the changes to the remote repository
git merge branch_name -> Merge a branch into the current branch
git pull origin branch_name -> Pull the changes from the remote repository
git fetch origin branch_name -> Fetch the changes from the remote repository
git clone url -> Clone a repository
